# 🧮 Calculator App

A modern and responsive Calculator application built using HTML, CSS, and JavaScript. This project was developed as part of my Frontend Development Internship at CodeAlpha.

## 📌 Overview

The Calculator App is designed to perform basic arithmetic operations with a clean and intuitive user interface. The application combines functionality with modern design elements to provide a smooth user experience.

## ✨ Features

* Addition, Subtraction, Multiplication, Division, and Modulus operations
* Real-time expression display
* Clear (AC) button to reset calculations
* Backspace functionality to remove the last entered character
* Keyboard support for faster input
* Responsive design for different screen sizes
* Interactive button animations and hover effects
* Error handling for invalid calculations

## 🎨 User Interface

The project features a modern glassmorphism-inspired design with:

* Transparent glass effect calculator panel
* Animated background elements
* Smooth transitions and hover interactions
* Clean and user-friendly layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JavaScript)

## 📚 Concepts Practiced

Through this project, I strengthened my understanding of:

* DOM Manipulation
* Event Handling
* Keyboard Events
* CSS Animations
* Responsive Web Design
* JavaScript Functions and Logic Building
